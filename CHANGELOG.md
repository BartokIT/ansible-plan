# Changelog

## Unreleased

### Fixed
- **Strategy Visualization:** Corrected the data structure for workflow strategy visualization.

### Added
- **REST API:** Introduced a REST API to expose plan execution as a service.
- **Jinja2 Templating:** Implemented Jinja2 support for dynamic playbook and variable rendering.
- **Ansible Vault Integration:** Added support for Ansible Vault to manage sensitive data, including a helper script for vault interaction.
- **Project Packaging:** Added `pyproject.toml`, `setup.cfg`, and `setup.py` to formalize the project packaging and installation process.
- **Example Workflows:** Included a comprehensive set of new example YAML files (`v1_complete_format.yml`, `sub_templating.yml`, etc.) to demonstrate various features and use cases.
